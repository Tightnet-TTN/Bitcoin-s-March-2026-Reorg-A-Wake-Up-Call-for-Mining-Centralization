This article dives into a rare two-block reorg that occurred on March 20, 2026, in Bitcoin's blockchain, triggered by Foundry, the largest mining pool at the time. It explains how this event sheds light on the growing issue of mining centralization, where a few dominant pools control most of Bitcoin’s hash rate. The piece emphasizes the risks this centralization poses to Bitcoin’s decentralization, security, and long-term viability, urging the community to take action before these issues become more widespread.
